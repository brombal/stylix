# Stylix Demos

These demos are to develop and test functionality. They aren't meant to be exhaustive tests or be
full demonstrations. They are just a useful way to playground features.

## Dynamic styles

Basic demonstration of styles using state variables.

<Preview>
  <Story name="Dynamic styles">
    {() => {
      const [color, setColor] = React.useState('blue');
      return (
        <>
          <$ color={color} border="1px solid green">
            {color} text
          </$>
          <select value={color} onChange={(e) => setColor(e.target.value)}>
            <option>red</option>
            <option>blue</option>
            <option>green</option>
          </select>
        </>
      );
    }}
  </Story>
</Preview>
